VScode添加右键 open with code 功能

发表于 2020-09-29

由于第一次安装的时候没留意，一直下一步导致漏勾选了”open with code”

如何挽救呢？

方法一：卸载重装/下载最新安装包直接安装
方法二：修改注册表

重装的方法相信大家都知道，就直接上方法二

修改注册表

  1. 文件后缀名为reg

  2. 路径为vscode的绝对路径，注意路径的\,在字符串要用\\(两个反斜杠)转义

  3. 保存之后，双击运行reg文件，会遇到一个提示，点击 “是”就可以了

  1. 接下来就可以用右键”open with code”测试一下了，另外命令行输入code .就可以用vscode打开当前文件夹了
