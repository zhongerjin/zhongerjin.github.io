  • Vue 采用双向数据流，React 采用单向数据流

  • JSX 是由 React 推出的，所以在 Vue 中不能使用

  • Vue 和 React 不需要配合 NPM 也能使用

关于cookie和session描述错误的是(A C D)

  • 通常下cookie比session安全

  • cookie数据存放在客户的浏览器上，session数据放在服务器

  • session是有有效期的，而cookie则没有有效期

  • session可以在多个服务器之间共享

  1. session是存在服务器上的，session Storage才是存放在客户的浏览器上
  2. cookie 可以多个服务器共享
  3. cookie 可以设置过期时间
  4. cookie可以被盗用，所以不安全

第十一题

修改下列哪个CSS 属性会可能触发回流？(C)

  • background-color

  • outline-width

  • border-width

  • transform

  1. 回流：元素的尺寸、结构发生改变时，浏览器重新渲染部分或全部文档的过程被称为回流。每个页面至少需要一次回流，就是在页面第一次加载的时候。
  2. 重绘：当页面中元素样式的改变不会影响它在文档流中的位置时，属性只影响元素的外观、风格，而不影响布局，浏览器将新样式赋予给元素并重新绘制它，这个过程称为重绘.
  3. 回流必将引起重绘，而重绘不一定会引起回流。

小米2021前端第二场笔试

第一题

使用浏览器访问http://miui.com/时，以下哪些过程发生在浏览器端？(A D)

  • 构建HTTP 请求

  • 解析HTTP 请求

  • 发送HTTP 响应

  • 渲染页面

  1. 浏览器构建HTTP请求，并发送
  2. 服务器端收到HTTP请求，并进行解析
  3. 服务器端发送HTTP响应
  4. 浏览器接收到响应，并进行页面渲染

第二题

下列哪些关于HTTP/2 的描述是正确的？(B C D)

  • 解决了TCP 队头阻塞的问题

  • 解决了浏览器线头阻塞的问题

  • 使用了头部压缩

  • 使用二进制分帧封装HTTP 消息

  1. 支持多种数据格式，增加POST、HEAD等方法，增加头信息，每次只能发送一个请求（无持久连接）
  2. HTTP/1.1：默认持久连接、请求管道化、增加缓存处理、增加Host字段、支持断点分块传输等
  3. HTTP/2.0：二进制分帧、多路复用、头部压缩、服务器推送

第六题

下面关于transform说法正确的是(A D)

  • 只对受控于盒模型的元素生效

  • 变形的原点默认是元素的左上角

  • 变形后元素在页面上的占位也随之变化

  • 可以通过matrix函数整合多种变形效果

  1. 变形的远点默认是元素的中心
  2. 变形后占位不会变化，它会创建一个新的图层来显示
  3. transform对行内元素不生效

第十一题

关于进程，下列说法错误的是？(C)

  • 进程有独立的地址空间

  • 进程间可以通过网络通信

  • 内存不能在不同进程间共享

  • 进程是系统进行资源分配和调度的基本单位

  1. 进程是操作系统资源分配的基本单位，而线程是任务调度和执行的基本单位

第十二题

下面哪个关于TCP 与 UDP 的描述是错误的？(D)

  • TCP 提供可靠度的字节流服务

  • 建立TCP 连接需要三次握手

  • 使用UDP 传输数据，源端和终端不建立连接

  • HTTP 建立在 UDP 之上

  1. Http3使用了UDP

第十三题

关于浏览器缓存，下列说法正确的是？(D)

  • Service Worker cache 的优先级没有 disk cache 高

  • 服务端没法控制浏览器缓存的过期时间

  • 浏览器不会将缓存存在内存中

  • 浏览器会先在浏览器缓存中查询是否有要请求的文件满足需求，再决定是否发送请求

  1. 优先级：service Worker -> Memory Cache -> Disk Cache -> Push Cache

第十六题

下列关于GBK和UTF8编码描述错误的是？(B)

  • GBK编码汉字占用2个字节

  • UTF8编码占用3个字节

  • GBK编码有包含中文简体和繁体字符

  • UTF8编码有包含中文和繁体字符

  1. GBK：中文、英文、数字均使用双字节来表示
  2. UTF-8：汉字占3个字节、数字占1个字节、英文字母各占一个字节
